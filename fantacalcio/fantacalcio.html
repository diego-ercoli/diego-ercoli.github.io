Presentation of the Platform:


As mentioned, in the last slide, some optimization problems in this platform are NP problems and ad-hoc solutions need to be studied. A problem, which is a particular version of the Knapsack one, is worth to be described.
Assume a list of n football players provided by Gazzetta dello Sport. Each player is assigned one of the following positions:
  • Goalkeeper (P)
  • Defender (D)
  • Midfielder (C)
  • Forward (A)
Additionally, each player has a specific price (or cost) w, updated by the newspaper after each official Serie A matchday. Moreover, each player is associated with a certain value v, computed as the average of their performance ratings throughout the season. Given a certain budget, the objective is to purchase exactly a certain number of players for each position, aiming to maximize the overall value of the assembled team.
See the next slides for a detailed formulation of the solution has been provided based on Dynamic Programming.
